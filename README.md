# Workload Evaluator

Веб-приложение для оценки загруженности подразделений предприятия и перераспределения задач.

Проект разработан по заданию для ИП Брюхачев Александр Александрович и закрывает требования по проектированию информационной системы, разработке интерфейса, тестированию, сопровождению, SEO-подготовке, сбору статистики и аудиту безопасности.

## Возможности

- регистрация и вход пользователей;
- разграничение ролей пользователя и администратора;
- просмотр подразделений, задач и общей статистики;
- создание задач и подразделений администратором;
- ручное перераспределение задач между подразделениями;
- автоматическое перераспределение задач из перегруженных подразделений;
- журнал активности;
- сбор статистики работы приложения;
- health-check, резервная выгрузка и аудит безопасности;
- SEO-файлы `robots.txt` и `sitemap.xml`.

## Технологии

- Node.js
- Express
- HTML
- CSS
- JavaScript
- Jest
- Supertest

## Запуск

Установить зависимости:

```powershell
npm install
```

Запустить сервер:

```powershell
npm start
```

Открыть приложение:

```text
http://localhost:3000
```

Если порт `3000` занят:

```powershell
$env:PORT=3001
npm start
```

## Данные для входа

Администратор по умолчанию:

```text
Пароль: secret
```

## Тестирование

```powershell
npm test
```

## Основные API

- `GET /api/health` - проверка работоспособности.
- `GET /api/departments` - список подразделений с загрузкой.
- `GET /api/tasks` - список задач.
- `GET /api/stats` - общая статистика.
- `GET /api/analytics` - статистика работы приложения для администратора.
- `GET /api/security-audit` - список мер безопасности для администратора.
- `GET /api/backup` - резервная выгрузка для администратора.
