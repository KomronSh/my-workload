# Размещение и продвижение

## Размещение

Приложение можно разместить на Node.js-хостинге, VPS или учебном сервере.

1. Установить Node.js.
2. Скопировать проект на сервер.
3. Выполнить `npm install`.
4. Задать переменные окружения при необходимости:
   - `PORT` - порт приложения.
   - `DATA_PATH` - путь к файлу данных.
   - `CORS_ORIGIN` - разрешенный origin для CORS.
5. Запустить `npm start`.
6. Настроить домен и HTTPS на уровне хостинга или reverse proxy.

## SEO-подготовка

- Основная страница содержит meta description, keywords и canonical.
- Страница входа закрыта от индексации через `noindex, nofollow`.
- Добавлены `public/robots.txt` и `public/sitemap.xml`.
- Описание приложения использует ключевые фразы: оценка загруженности подразделений, перераспределение задач, управление задачами.

## Продвижение

- Опубликовать приложение на домене с HTTPS.
- Добавить ссылку на приложение в учебное портфолио или карточку проекта.
- Подготовить краткое описание назначения и сценариев использования.
- Отслеживать статистику через административный раздел "Статистика работы приложения".
- Улучшать тексты страниц по результатам анализа поисковых запросов и поведения пользователей.
